# Format Library

## Car Founder UGC Proof

Best for:

- solo owners
- service businesses
- missed calls
- booking/follow-up

Structure:

1. Physical prop hook in car.
2. One line: the owner missed something.
3. Product proof insert.
4. Tiny relieved payoff.

Example line:

> I missed the call. Clover didn't.

## Office Sticky-Note Chaos

Best for:

- founder/operator workplace humor
- lead response
- inbox overload

Structure:

1. Laptop/phone chaos visible.
2. Physical object hits or interrupts.
3. Performer realizes Clover already handled it.
4. Quick proof insert.

Risk:

Generated sticky-note and screen text can break the clip. Use icon-only or deterministic overlays.

## Mascot Dealer Table

Best for:

- recurring Clover show
- strategy/comedy
- quotable founder truths

Structure:

1. Card already moving.
2. Bad option cards are dealt.
3. Clover slaps the obvious winning card.
4. One tiny line.

Example line:

> Stop posting. Start casting.

## Support Desk Mascot

Best for:

- inbound lead chaos
- client ops
- agency/workplace comedy

Structure:

1. Phone lines light up.
2. Clover headset clicks on.
3. One impossible support request lands.
4. Clover calmly solves the wrong problem behind the problem.

## Clover Sitcom Episode (PROVEN — Ep. 1-3)

Best for:

- recurring character/show content
- workplace truths founders quote at each other
- brand IP compounding across episodes

Structure (single 15s generation, 3 shots in-prompt):

1. Oner push through an escalating ensemble problem — each cast member lands one beat as the
   camera passes (this staging IS the one-mouth-at-a-time enforcement).
2. Settle on the calm one. Hard cut: deadpan question.
3. Hard cut: quotable reveal + a lighting change tied to the final words.

Example payoff: "The demo was an hour ago. Clover ran it. They signed."

Locked assets: character sheet (Ep. 2+), voice map (Ep. 1+). See `examples/clover-demo-ep3/`.

## Hybrid New Hire (PROVEN — v0.1 launch)

Best for:

- product launches and ability showcases
- "real person + real desk" feed-native credibility with mascot IP on top
- any feature that can be shown as object interaction instead of UI

Structure (single 12s generation, 3 shots in-prompt):

1. Photoreal founder mid-overwhelm (physical mess hook), one setup line listing the pains.
2. Desk-level macro: the mute animated mascot fixes each pain as a silent physical-comedy
   beat (tap, sweep, flick) — 2-3 abilities max, screens stay abstract.
3. Founder payoff line + mascot button (thumbs-up, mug sip — series callback).

Rules: human carries ALL dialogue (with DIALOGUE LOCK), mascot stays mute, brand avatar as
start-frame ref. See `examples/clover-v01-launch/` and `prompts/seedance-prompt-blocks.md`.
