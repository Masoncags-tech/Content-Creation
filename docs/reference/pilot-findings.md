## Pilot 013: UGC Reference Calibration

Finding:

The stronger growth-video lane is real creator UGC with one physical prop gag and one product proof insert.

Generated candidates:

- `pilot-013a-car-seat-founder`: good structure, bad generated end-card typo.
- `pilot-013b-office-sticky-note`: strongest visual chaos, too much readable generated text.
- `pilot-013c-car-seat-short-no-text`: best current direction: cleaner hook, cleaner continuity, fewer text artifacts.

Decision:

Use real creator/product-proof videos for growth testing. Keep the mascot as IP for recurring show formats, not as the only visual surface.

## Important Nuance

The mascot is not the problem.

The problem is abstract flashing screens, boxes, captions, and motion that do not create a believable scene. Mascot content works only when it has:

- a repeatable set
- a physical action
- a native voice
- a clear joke/workplace truth
- cuts that feel like coverage from one scene
