# Creative Elements

Use this as the menu before writing prompts.

These are reusable patterns and examples, not fixed creative mandates. Load the pattern, then adapt the setting, prop, and proof method to the venture.

## Physical Hooks

| Hook | Best format | Risk |
| --- | --- | --- |
| Coffee lid becomes mic | Car UGC, desk UGC | Can feel random unless tied to the line. |
| Sticky note hits face | Office mockumentary | Generated text artifacts. Use blank note or post overlay. |
| Phone buzz stack | UGC, screenlife | Fake notification text. Use icons/bubbles only. |
| Coffee almost spills | Founder/operator UGC | Needs visible reaction. |
| Headset click | Mascot support desk | Needs native voice and room tone. |
| Card slap | Mascot dealer table | Needs consistent table/set design. |
| Laptop alert glow | Office UGC | Screen text must be deterministic or unreadable. |
| Calendar checks appear | Proof insert | Better rendered in Remotion/HTML. |

## Proof Inserts

Proof inserts should be visual, not text-heavy.

Use:

- green check icons
- calendar slot fills
- task card moves to done
- call row gets answered
- invoice moves to sent
- booking tile appears
- message bubble gets a checkmark

Avoid:

- generated paragraphs
- fake client names
- fake logos
- screenshots with model-written text
- claims that need legal proof

## Settings

| Setting | Use when | Notes |
| --- | --- | --- |
| Parked car | Owner is between jobs, errands, or meetings. | Strong native creator feel. |
| Messy desk | Inbound/admin chaos. | Good for sticky notes, phone, coffee, laptop. |
| Front office | Local business trust. | Avoid fake patient/client info. |
| Dealer table | Mascot strategy/comedy. | Repeatable IP set. |
| Support desk | Mascot ops/comms. | Great for phone/headset voice. |
| Podcast chair | Founder truth / quotable clips. | Needs sharp writing. |

## Creative Pairings

Strong combinations:

- parked car + missed call + calendar check
- messy desk + sticky note + lead already replied
- support desk + headset click + phone lines calm down
- dealer table + bad option cards + one winning card
- front office + appointment card + task moves to done

Weak combinations:

- abstract flashing boxes + captions
- random mascot in blank room + voiceover
- generated dashboard as hero
- long product pitch over unrelated footage
- fake UI text as the central proof

## Clover Mascot Use

The mascot works when it has a job.

Good mascot roles:

- dealer of strategic decisions
- support-desk operator
- tiny podcast guest
- night-shift operations employee
- workflow referee

Bad mascot roles:

- generic dancing object
- floating explainer icon
- random caption narrator
- substitute for the whole scene
