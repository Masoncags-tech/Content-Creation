# Script Lab

This is the working room for making the words sharper before generating video.

## The Rule

A good short script is not a miniature ad. It is a filmed moment with a line worth repeating.

## Script Score

Score each concept 0-5:

| Category | Question |
| --- | --- |
| Recognition | Does the target viewer immediately know this situation? |
| Physicality | Can the first second happen without captions? |
| Compression | Is the line short enough to survive social pacing? |
| Specificity | Is the workflow concrete? |
| Payoff | Is there a quotable turn or satisfying proof? |

Kill anything under `18/25` before generation.

## Line Patterns

### Missed Work

Use when the owner failed to catch something and Clover handled it.

- "I missed the call. Clover didn't."
- "I forgot the follow-up. Clover didn't."
- "I left them on read. Clover booked it."
- "I was driving. Clover closed the loop."

### Owner Relief

Use when the emotional payoff is relief, not hype.

- "That used to be my whole afternoon."
- "This is the first quiet Monday I've had."
- "I did one thing. It handled the boring five."
- "I didn't need a dashboard. I needed it done."

### Founder Truth

Use for Mason/founder POV or mascot show content.

- "If the work needs babysitting, it is not handled."
- "The first workflow should be boring."
- "Stop posting features. Show the mess disappearing."
- "The demo is not the product. The done work is."

### Mascot Punchline

Use when Clover is a character inside the scene.

- "Stop posting. Start casting."
- "That is not strategy. That is panic with a calendar invite."
- "I found the bottleneck. It was you."
- "Good news: the system works. Bad news: it has opinions."

## Rewrite Ladder

Start factual, then compress into a line.

| Draft | Better | Best |
| --- | --- | --- |
| Clover Digital handles missed calls and follow-ups for small businesses. | I missed the call. Clover followed up. | I missed the call. Clover didn't. |
| Clover can help business owners save time on admin work. | It handled the boring part while I was busy. | I did one thing. It handled the boring five. |
| Clover is not just a chatbot. | It does the work after the message. | If it only chats, it is not hired. |

## Hook Before Script

Do not write the line until the hook exists.

Bad order:

1. Write pitch.
2. Invent visuals.

Good order:

1. Pick physical hook.
2. Pick workflow proof.
3. Write the shortest line that connects them.

## Script QA

Before generation, every script must answer:

- What moves at second zero?
- What object creates the line?
- What proof appears on screen?
- What is the quote?
- What can be removed?
