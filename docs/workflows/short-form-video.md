# Short-Form Video Workflow

## Purpose

Create TikTok/Reels/Shorts clips that feel native to the feed instead of generated for a demo reel.

## Format First

Start with a format before writing a script:

- UGC confession
- product-in-life microdemo
- workplace mockumentary
- POV operator moment
- screenlife panic
- mascot show scene
- cinematic cold open

Do not begin from "make a viral AI video." That produces random motion.

## The 10-Second Default

Use 8-12 seconds until the mechanics are consistently good.

| Time | Beat | Requirement |
| --- | --- | --- |
| `0.0-1.0s` | Physical hook | A face, hand, prop, camera move, mess, notification, or object gag is already active. |
| `1.0-5.5s` | Situation line | One human-readable problem or claim, spoken in-scene. |
| `5.5-7.5s` | Proof insert | Phone, calendar, receipt, dashboard, task list, or object proof. Avoid readable fake text. |
| `7.5-10.0s` | Payoff | Reaction, reversal, quote, or clean hold frame. |

## Variant Rules

Generate 2-4 variants per run. They must differ in the first-second visual, not only copy.

Useful first-second variants:

- prop drops into hand
- phone violently buzzes
- sticky note hits face
- coffee almost spills
- laptop alert flashes
- headset clicks on
- card slaps table

## Transition Rules

A transition is not an effect. It is a reason to be in the next shot.

Valid cut reasons:

- a new speaker starts
- a reaction lands
- a prop becomes important
- a phone/product proof insert is needed
- the punchline needs a closer angle
- audio starts before picture changes

Avoid:

- flash transitions
- zoom blur to hide discontinuity
- cross dissolves between unrelated generated shots
- cutting to a new location without a setup
- two shots in a row that start with a centered idle character

## Output Requirements

Each run should produce:

- final MP4 or generated job URL
- contact sheet
- prompt
- script
- performance map
- QA report
- score or note explaining why scoring was unavailable
- next iteration decision
